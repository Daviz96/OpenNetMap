# Prompt sorgente

Questa cartella raccoglie i **prompt** usati per generare ed evolvere OpenNetMap.
Sono materiale di riferimento storico/progettuale, non descrizioni dello stato corrente.

| File | Scopo |
|---|---|
| `network_inventory_prompt.md` | Prompt iniziale: sviluppo del LAN scanner / network inventory tool. |
| `network_inventory_enterprise_prompt.md` | Prompt per l'evoluzione verso una piattaforma enterprise. |
| `topology_engine_prompt.md` | Prompt per il Topology Engine (pianificato come Sprint 7-8). |

Per lo stato corrente del progetto vedi `docs/Claude docs/STATO_PROGETTO.md` e `README.md`.
